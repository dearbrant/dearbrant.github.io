选择满足自己阅读需要的电子书阅读器

电子书的来源决定了阅读器的选择。我目前主要在用微信读书，书库的覆盖面不错，移动端 App 设计利落体验友好，图书商店的营销推广也比较克制，让人能专心阅读。

微信读书没有官方阅读器硬件，它提供墨水屏版本的软件安装包。我甄别对比了几家使用开放系统的产品，考量点依次为 核心操作的流畅度 > 屏幕的清晰度 > 做工和价格，最后选择了 文石 BOOX Poke 3。如果不介意做工，博阅 Likebook P6 也是个选项，便宜一半。

优化硬件配置，更新软件使用习惯，让阅读更沉浸

我用阅读器只看书，听书 / 听歌 / 上网 都不用，甚至背光灯也不需要，没光源就不看了，当它是本纸质书。明确了自己的需要，就可以着手优化工具的软硬件，让工具更称手，改造工具的同时也会让你的使用习惯得到更新。

以 BOOX Poke 3 为例，我的优化思路是删繁就简 1. 去除不需要的功能 2. 简化界面 3. 省电 4.改造微信读书的使用习惯，具体步骤如下：

应用

1. 在中文应用商店安装微信读书
2. 冻结一切可冻结的应用
3. 将其他应用全部收在文件夹里，界面只剩下 微信读书 + 文件夹 两个

设置

1. 升级固件
2. 应用设置 - 关闭系统自带的图书商店
3. 更多设置 - 让底部导航只展示 icon
4. 电量设置 - 以省电好用为出发点 禁止关机 / 3 分钟休眠 / 开机时 Wi-Fi 保持关闭
5. 语言设置 - 调整成英文，视觉上更简化，界面显得陌生也是种简化手段

快捷下拉菜单

1. 关闭悬浮球
2. 调整合适的对比度
3. 关闭背景灯功能
4. 快捷菜单里仅保留 悬浮球 / 对比度 / 刷新模式 / Wi-Fi

改造微信读书的使用习惯

0. 改造的思路是在手机上做书籍和笔记的管理，在电子书里离线阅读
1. 在手机上打开微信读书，整理好分组，把要读的书留在分组外面，5 本内最佳
2. 在阅读器上登录微信读书，账号和书籍分组状态都会同步过来
3. 将留在分组外需要阅读的书全部下载离线
4. 离线后，在下拉菜单里关掉 Wi-Fi，只读离线书籍，不要贪心
5. 书籍管理 / 新书的添加 / 笔记导出，都在手机上进行，阅读器保持定期同步
